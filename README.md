# ConvertWeight

A simple, user-friendly Java console application designed to handle weight conversions between different standard units (such as Kilograms and Pounds). This project demonstrates input validation, mathematical formatting, and practical application of conversion formulas in Java.

## Features

- **Bi-directional Conversion:** Easily convert weight from Kilograms (kg) to Pounds (lbs) and vice versa.
- **Precision Formatting:** Formats output values to a clean decimal threshold for high readability.
- **Input Validation:** Gracefully handles invalid choices or non-numeric inputs to prevent application crashes.
- **Looping Interface:** Allows users to perform multiple conversions in a single session without restarting the program.

## Formulas Used

The application applies the standard mathematical conversion ratios:

- **Kilograms to Pounds:** $Pounds = Kilograms \times 2.20462$
- **Pounds to Kilograms:** $Kilograms = \frac{Pounds}{2.20462}$
